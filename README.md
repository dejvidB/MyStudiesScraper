## My Studies Scraper


__Website link__: https://my-studies-scraper.live


### Project Overview

Σκοπός αυτού του project είναι ο εκσυγχρονισμός της [ιστοσελίδας της Ηλεκτρονικής Γραμματείας του ΕΚΠΑ](https://my-studies.uoa.gr/),
ώστε να ανταποκρίνεται στα δεδομένα και τις ανάγκες του σύγχρονου φοιτητή, αλλά
και να τον βοηθήσει προσφέροντάς του τις πιο χρήσιμες δυνατότητες μαζεμένες σε μία mobile-friendly και φιλική προς τον χρήστη ιστοσελίδα.

#### Features

- Εμφάνιση όλων των μαθημάτων, οργανωμένων σε κατηγορίες/κατευθύνσεις

- Σημειωμένα όλα τα περασμένα και χρωστούμενα μαθήματα ανά εξάμηνο, καθώς επίσης σημειώνεται
και ο βαθμός του κάθε μαθήματος

- Εμφάνιση γενικών πληροφοριών για τον φοιτητή, όπως έχουν περαστεί από τη γραμματεία
του τμήματός του και αναγράφονται στις αντίστοιχες υπηρεσίες

- Εμφάνιση Μέσου Όρου, υπολογίζοντας όλα τα περασμένα μαθήματα μέχρι εκείνη τη στιγμή
και συνυπολογίζοντας τη βαρύτητα (ECTS) του καθενός

- Tο ιστορικό των δηλώσεων σε κάθε εξεταστική και οι βαθμοί ταξινομημένοι
ανά εξεταστική περίοδο

- Δήλωση μαθημάτων (Η λειτουργία αυτή γίνεται αυτόματα διαθέσιμη και ορατή μόνο κατά την περίοδο δήλωσης μαθημάτων)

### Disclaimer

Όλα όσα αναφέρονται παραπάνω ή στο ίδιο το site, εξηγούνται στα αρχεία κώδικα που υπάρχουν
σε αυτό το repository. Όλα τα στοιχεία που εμφανίζονται στην ιστοσελίδα, συλλέγονται από
τα αντίστοιχα επίσημα sites του πανεπιστημίου με μοναδικό σκοπό την καλύτερη και πιο σύγχρονη παρουσίασή τους.  
__Σε καμία περίπτωση δεν συλλέγεται οποιοδήποτε στοιχείο αναγράφεται
ή υπάρχει δηλωμένο στα προσωπικά στοιχεία του φοιτητή.__  
Η μόνη πληροφορία που βρίσκεται σε βάση δεδομένων, στην οποία όλοι έχουν δικαίωμα ανάγνωσης, είναι η αντιστοιχία `κωδικών μαθημάτων` -> `ECTS`, μόνο για τα τμήματα που χρησιμοποιούν ECTS, για τον υπολογισμό του Μέσου Όρου των φοιτητών.
